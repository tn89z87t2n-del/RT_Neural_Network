Built from scratch — no ML libraries. The neural network (forward pass, manual backpropagation, SGD / Momentum / Adam) is implemented in vanilla JS with Float32Array weights.
Dataset: a learnable MNIST-style set is generated procedurally on the fly by rendering the glyphs 0–9 to an offscreen canvas with random rotation, scale, translation, stroke jitter and noise.
